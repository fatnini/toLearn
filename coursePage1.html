在這門課裡你將學到什麼？

  • 網頁設計，基礎知識
  • HTML 語法，概念實作能力
  • CSS , Javascript 概念與知識
  • 能做出一個基礎的 HTML 網頁

什麼是 HTML 與 CSS，為什麼要打造網頁，就一定要先學它們？

HTML元素是構建網站的基石。HTML允許嵌入圖像與物件，並且可以用於建立互動式表單，它被用來結構化資訊——例如標題、段落和列表等等，也可用來在一定程度上描述文件的外觀和語意。 而CSS是一種用來為結構化文件（如HTML文件或XML應用）添加樣式（字型、間距和顏色等）的電腦語言。CSS 能夠對網頁中元素位置的排版進行像素級精確控制，支援幾乎所有的字型字號樣式，擁有對網頁物件和模型樣式編輯的能力。

當你掌握了HTML與CSS，就可以開始規劃並建置你想要的網頁內容 !
